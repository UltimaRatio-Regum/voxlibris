# Getting Started with VoxLibris

VoxLibris transforms plain text into expressive audiobooks using advanced AI-powered text-to-speech technology. This guide will walk you through creating your first audiobook.

## First Login

When you first access VoxLibris, you'll see a login screen. Log in with the administrator credentials provided by your system administrator.

> **Important:** Change your password after your first login by clicking your username in the top-right corner and selecting **Change Password**.

## Interface Overview

After logging in, you'll see the main interface with these tabs:

| Tab | Purpose |
|-----|---------|
| **Project Wizard** | Create new audiobook projects with a guided step-by-step flow |
| **Projects** | Browse and edit your existing audiobook projects |
| **Jobs** | Monitor audio generation and export jobs |
| **Settings** | Configure TTS engines, voices, and parsing settings |
| **Users** | *(Admin only)* Manage user accounts and registration |

## Creating Your First Audiobook

The fastest way to create an audiobook is through the **Project Wizard**:

1. **Upload your text** — Paste text directly or upload a `.txt` or `.epub` file
2. **Wait for analysis** — The AI will automatically detect speakers, assign emotions, and segment the text into chunks
3. **Assign voices** — Choose voices for each detected speaker (narrator, characters, etc.)
4. **Generate audio** — Start the generation process and monitor progress in real-time

Once generation completes, you'll be taken to the **Project Editor** where you can review, edit, and export your audiobook.
